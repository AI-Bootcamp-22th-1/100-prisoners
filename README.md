# **100-prisoners**

Solve the 100 prisoners problem.

## **The Problem**

- 100명의 수감자에게는 1번부터 100번까지 개별적으로 번호가 매겨져 있습니다.
- 밖에서는 보이지 않는 1부터 100까지 번호가 매겨진 불투명 서랍 100개가 있는 찬장이 있는 방.
- 1부터 100까지 숫자가 적힌 카드들이 무작위로 한 장씩 서랍에 들어가고, 모든 서랍은 닫힌 상태로 게임 시작이 진행된다.
- 죄수들은 방 밖에서 출발합니다.
  - 그들은 누군가 방에 들어가기 전에 몇 가지 전략을 세울 수 있습니다.
  - 수감자들은 한 명씩 방에 들어가 서랍을 열고 서랍 안의 카드 번호를 확인한 후 서랍을 닫을 수 있습니다.
  - 수감자는 서랍을 50개까지만 열 수 있습니다.
  - 한 죄수가 자신의 죄수 번호를 찾으려고 애쓴다.
  - 자신의 번호를 발견한 죄수는 다른 죄수들과 격리됩니다.
- 만약 100명의 죄수 **모두**가 자신의 번호를 찾으면 모두 사면될 것입니다. 만약 한 명이라도 찾지 못하면 모든 형벌은 그대로 유지됩니다.


## **The task**

1. 죄수들이 무작위로 서랍을 여는 게임을 수천 번 시뮬레이션합니다.
2. 위키피디아 문서에 언급된 최적 전략을 죄수들이 사용하는 게임을 수천 번 시뮬레이션하십시오.
  - 먼저 겉면에 그의 죄수 번호가 적힌 서랍을 엽니다.
  - 안에 있는 카드에 자신의 숫자가 적혀 있으면 성공이고, 그렇지 않으면 공개된 카드와 같은 숫자가 적힌 서랍을 엽니다.
    (최대 개수까지 열 수 있습니다).

이 페이지에서 두 전략의 성공 확률을 계산하여 비교하십시오.
